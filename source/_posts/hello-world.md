# 欢迎来到我的博客

这是我的第一篇博客文章，很高兴能与你分享！

## 博客搭建历程

一直以来都想拥有一个属于自己的博客，用来记录学习笔记、分享技术心得、沉淀生活感悟。经过多方调研和对比，最终选择了 Hexo + Butterfly 的组合来搭建这个博客。

### 为什么选择 Hexo？

Hexo 是一个快速、简洁且高效的博客框架，具有以下优势：

1. **快速生成**：Hexo 使用 Node.js 开发，生成静态页面的速度非常快
2. **Markdown 支持**：支持 Markdown 语法，写作体验极佳
3. **丰富的主题**：拥有大量优秀的主题可供选择
4. **插件生态**：插件丰富，功能扩展性强
5. **一键部署**：支持一键部署到 GitHub Pages、Vercel 等平台

### 为什么选择 Butterfly 主题？

Butterfly 是一款基于 Hexo 的优秀主题，具有以下特点：

1. **简洁美观**：界面设计简洁优雅，阅读体验极佳
2. **功能完善**：支持暗黑模式、评论系统、搜索功能等
3. **响应式设计**：完美适配各种设备
4. **高度可定制**：提供丰富的配置选项
5. **持续更新**：作者积极维护，不断优化

## 博客功能介绍

### 1. 文章展示

博客支持 Markdown 语法，可以轻松编写文章，支持：

- 代码高亮
- 图片展示
- 表格插入
- 数学公式
- 流程图

### 2. 分类与标签

文章可以按分类和标签组织，方便读者查找感兴趣的内容。

### 3. 评论系统

集成了 Giscus 评论系统，基于 GitHub Discussions，完全免费且稳定可靠。

### 4. 搜索功能

支持全站搜索，快速定位目标文章。

### 5. RSS 订阅

提供 RSS 订阅功能，读者可以订阅博客更新。
