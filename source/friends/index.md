# 友情链接

欢迎交换友链！请按照以下格式留言：

## 我的友链信息

- 博客名称：你的博客名称
- 博客地址：https://yourusername.github.io
- 博客头像：https://yourusername.github.io/images/avatar.png
- 博客描述：一个简洁优雅的个人博客

## 友链列表

按照以下格式添加友链：

```yml
- name: 友链名称
  link: https://example.com
  avatar: https://example.com/avatar.png
  descr: 友链描述
```

## 友链申请要求

1. 博客内容积极向上，无不良信息
2. 博客可以正常访问
3. 优先考虑技术类博客
4. 申请前请先添加我的友链
